Introduction

GitHubAPI.jl

Author/Maintainer: José Bayoán Santiago Calderón (Nosferican)

This application is developed by the Social and Decision Analytics Division of the Biocomplexity Institute and Initiative, University of Virginia.

GitHubAPI.jl is an application to collect GitHub data on open-source projects (i.e., commit data for repositories with OSI-approved licenses).

Prerequisites

  • Julia
  • Internet access
  • Access to the SDAD postgis_1/oss database (contact Aaron Schroeder)
  • A GitHub account
  • A GitHub personal access token
Warning

Currently the application assumes one is working locally. This will change once the containers are set up.

  • Check out the Public API section for documentation on how to use the application.

  • The Private API (Internals) provide useful information, but is not required for usage.

Note

While the application is meant for internal use. The code is ISC licensed and may be useful for other people. Do feel free to fork the project / re-use the code for your purposes.
